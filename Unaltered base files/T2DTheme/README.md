# T2DTheme

A dark Aurora theme.

## Details

Changes include:

1) Black background
2) White text
3) Font selection options
4) Custom icons (thanks to AGM-114)

![T2DTheme](/T2DTheme.png?raw=true)

## Dependencies

Requires AuroraPatch >= 0.1.0, Lib.dll from AuroraPatch, and the ThemeCreator >= 1.3.0.0 patch.
